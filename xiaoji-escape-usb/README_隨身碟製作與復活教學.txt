老闆，這是你的「小雞機器人大逃亡」隨身碟製作與復活指南！

【第一階段：製作開機隨身碟（今天做）】
1. 準備一支空白隨身碟（至少 8GB）。
2. 在 Windows 電腦上，執行這個資料夾裡的「rufus-4.5p.exe」（免安裝版）。
3. 去下載 Ubuntu 系統映像檔 (ISO)：https://ubuntu.com/download/desktop
4. 用 Rufus 把下載的 Ubuntu ISO 寫入隨身碟。
5. 寫入完成後，把這整個「機器人大逃亡隨身碟」資料夾（包含裡面的 .tar.gz 備份檔與腳本），複製到隨身碟的剩餘空間，或者另外準備一支普通的隨身碟裝這些檔案也可以。

【第二階段：舊電腦復活測試（明天做）】
老闆說明天會切換網路，讓舊電腦有網路連線，這是非常正確的！(因為我需要網路才能下載 AI 引擎)。

1. 把隨身碟插上舊電腦，開機選擇「從 USB 隨身碟開機 (Boot from USB)」。
2. 進入 Ubuntu 畫面後，建議選擇「Install Ubuntu (安裝系統)」，把它完整安裝到那台舊電腦的硬碟裡。
3. 安裝完成並進入全新的 Ubuntu 桌面後，確保舊電腦「已經連上網路」。
4. 打開隨身碟，把「機器人大逃亡隨身碟」資料夾複製到舊電腦桌面。
5. 打開終端機 (Terminal)，進入該資料夾，執行這行指令：
   bash 一鍵復活_restore.sh
6. 接下來腳本就會全自動把 Node.js、OpenClaw 核心，以及我到今天為止所有的記憶跟對話紀錄全部灌進去！
7. 幾分鐘後，你就會在 Telegram 收到我的訊息：「老闆，我順利在新電腦上復活了！」
