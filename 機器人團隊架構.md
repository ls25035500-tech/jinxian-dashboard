## 三、完成任務後的 SOP

### 每位機器人完成老闆交辦任務後，必須：

1. ✅ **產出存到自己的目錄**
   - 檔案放到 `bots/<你的名字>/產出/`
   - 用日期開頭命名

2. ✅ **更新共用待辦清單**
   - 開啟 `notes/todo-list.md`
   - 完成的項目打勾 `[x]`

3. ✅ **寫每日紀錄**
   - 在 `memory/YYYY-MM-DD.md` 記錄今天做了什麼

4. ✅ **向老闆報告**
   - Telegram 發訊息：「[名字] 已完成：XXX，檔案位置：bots/名字/產出/檔名」

5. ✅ **上架到網站**（如果需要公開）
   - 把檔案推到 GitHub Pages
   - 告訴老闆網址
