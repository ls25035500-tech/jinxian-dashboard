📌 重點摘要

  • Claude Cowork 是 Anthropic 2026 年推出的 AI 工作助理，與傳統 AI 不同在於「能幫你把工作做完」，不只是給答案
  • 需安裝桌面版 App（Windows/Mac），需 Claude Pro 訂閱（$20/月）
  • 核心概念：在指定的「資料夾」內工作，可存取本機檔案、操作瀏覽器與應用程式

⚙️ 核心功能

  • 📁 專案工作區 — 指定本地資料夾，AI 在安全範圍內讀寫檔案
  • 📄 文件生成 — 能產出 Word、Excel、PPT 等檔案，自動存到專案資料夾
  • 📱 Dispatch — 手機遠端指派任務給家中電腦的 Cowork
  • 🧠 CLAUDE.md — 專案記憶文件，記錄行為準則與偏好
  • 📅 排程任務 — 固定時間自動執行工作（如每週一製作簡報）
  • 👥 子代理（Subagent） — 多任務並行處理，最後彙整結果
  • 🔌 連接器（Connectors） — 串接 Canva、Google 日曆、Gmail 等外部工具
  • 🎯 Skills — 自訂 SOP 說明書，重複使用的工作流程
  • 🧩 Plugins — 把 Skills + Connectors 打包成套裝工具箱
  • 🖥️ Computer Use — 像真人操作電腦軟體（Keynote→PPT 轉檔等）

💡 實際應用範例

  • 📸 社區大學攝影講師用 Cowork 準備課程教材、管理學員名冊
  • 📊 手機翻拍紙本名冊 → Cowork OCR 辨識 → 自動產出 Excel
  • 📱 人在外面用手機指派：整理新學員名單、統計相機持有數
  • 🖼️ 學員繳交攝影作業 → Cowork 自動依主題分類到資料夾
  • 🏥 遠端幫父親在醫院網站掛號（瀏覽器操作）
  • 🔄 外地演講發現檔案格式不符 → 遠端 Cowork 幫忙轉檔

🆚 Cowork vs 傳統 AI

  • 傳統 AI：告訴你「答案是什麼」
  • Cowork：直接「幫你把工作做完」
  • 支援操作瀏覽器、編輯文件、管理檔案
  • 可排程、可分身（子代理）、可串接外部服務
