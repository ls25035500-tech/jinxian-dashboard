# 🦐 金仙遠端維修能力清單

> 給工程師看：USB 插下去、點兩下、傳 tcp:// 給金仙，以下全部可以遠端處理

---

## 🔌 連線方式

客戶點 USB 裡的 `start.bat` → 畫面出現 `tcp://xxx.run.pinggy-free.link:PORT` → 傳給金仙 → 金仙 SSH 登入

---

## 🏥 系統健康檢查（免動手，自動出報告）

| 檢查項目 | 金仙能做什麼 |
|:---|:---|
| CPU 型號 / 使用率 | 查出處理器規格、即時負載 |
| RAM 大小 / 使用量 | 抓出總記憶體、剩餘量 |
| 硬碟空間 | 每顆磁碟的總容量 / 已用 / 剩餘 |
| 開機時間 | 上次重開是幾天前 |
| Windows 版本 | 作業系統版號、更新狀態 |
| 網路 IP / DNS / 閘道 | 完整網路組態 |

---

## 🧹 系統清理

| 問題 | 金仙能做什麼 |
|:---|:---|
| C 槽爆滿 | 跑 `cleanmgr` 磁碟清理（暫存檔、更新備份、資源回收筒） |
| Temp 資料夾肥大 | 清空 `%TEMP%` |
| 瀏覽器快取 | 清除 Edge/Chrome 暫存 |
| Windows Update 快取 | 清 `C:\Windows\SoftwareDistribution` |

---

## 🖨️ 印表機問題

| 問題 | 金仙能做什麼 |
|:---|:---|
| 印表機不見 | 列出所有已安裝印表機 |
| 預設印表機跑掉 | 重新設定預設印表機 |
| 列印佇列卡住 | 清除列印佇列、重啟 Print Spooler 服務 |
| 驅動異常 | 檢查印表機驅動狀態 |

---

## 🌐 網路問題

| 問題 | 金仙能做什麼 |
|:---|:---|
| 網路磁碟機斷線 | 重新掛載 NAS / 共用資料夾 |
| DNS 解析異常 | `ipconfig /flushdns` 清除 DNS 快取 |
| IP 衝突 / DHCP 異常 | `ipconfig /release` + `ipconfig /renew` 重取 IP |
| 無法上網 | ping 閘道 / DNS / 外網，找出斷點 |
| 防火牆擋住 | 檢查 Windows 防火牆規則 |
| 網卡驅動異常 | 重啟網卡、檢查驅動狀態 |

---

## 💻 系統異常

| 問題 | 金仙能做什麼 |
|:---|:---|
| 電腦變慢 | 檢查 CPU/RAM 使用率，找出吃資源的程式 |
| 藍白畫面 / 當機記錄 | 查看事件檢視器錯誤日誌 |
| 開機啟動太多 | 列出開機自動啟動的程式 |
| Windows 服務異常 | 檢查/重啟特定服務 |
| 系統檔案損壞 | `sfc /scannow` 系統檔案檢查 |
| Windows 映像損壞 | `DISM /Online /Cleanup-Image /RestoreHealth` |

---

## 🔍 檔案問題

| 問題 | 金仙能做什麼 |
|:---|:---|
| 檔案找不到 | 全碟搜尋（支援萬用字元 `*.pptx`） |
| 誤刪檔案 | 檢查資源回收筒，嘗試復原 |
| 檔案權限異常 | 檢查檔案/資料夾權限 |
| 硬碟錯誤 | `chkdsk` 檢查磁碟錯誤 |

## 👥 帳號/權限

|:---|:---|
| 密碼忘記 | 重設本機帳號密碼 |

## 🔄 系統重啟 / 關機

| 動作 | 金仙能做什麼 |
|:---|:---|
| 遠端重開機 | `shutdown /r /t 0` |
| 定時重開 | `shutdown /r /t 3600`（1小時後） |
| 遠端關機 | `shutdown /s /t 0` |
| 取消排程重開 | `shutdown /a` |

## 🔒 安全說明

- 隧道 60 分鐘自動過期
- 金仙用專屬 `jxhelper` 帳號，用完可刪
- 不會修改客戶原有帳號密碼
- 不做任何未經授權的操作
- 關閉 `start.bat` 視窗 = 立即斷線
