# 🦐 金仙 USB 工具箱

> 插上去 → 點兩下 → 金仙連進來  
> 不用裝任何軟體，不用開 port，不用找 IP

---

## 🎯 原理

```
客戶電腦 (NAT/防火牆後面)
     │
     │  USB 插入 → start.bat
     ├─ 1. 自動開啟 Windows SSH Server
     ├─ 2. 自動設定防火牆規則
     └─ 3. SSH 反向隧道 → serveo.net
              │
              ▼
     金仙收到連線指令 → SSH 跳板連入 → 遠端維修 🛠️
```

因為是**出去**的連線（跟開網頁一樣），防火牆完全不擋。

## 🚀 使用方式

### 到客戶那邊

1. USB 插進客戶的 Windows 電腦
2. 打開 USB 目錄，對 `start.bat` **點兩下**
3. 如果跳出 UAC（使用者帳戶控制），按「是」
4. 畫面會跳出黑底視窗
5. 把螢幕上顯示的那行指令截圖或複製傳給金仙
6. **關掉視窗 = 斷線**，金仙就連不進去了

### 畫面顯示範例

客戶電腦螢幕會顯示：
```
Forwarding TCP connections from serveo.net:54321
```

### 金仙收到的連線指令

老闆把 port 號 + 客戶 Windows 帳密傳給金仙：
```
ssh -p 54321 sga@serveo.net
密碼: xxxxxx
```

金仙輸入這行就能直達客戶電腦。

## 🔒 安全性

| 項目 | 說明 |
|:---|:---|
| 傳輸加密 | SSH 加密 + serveo 加密 |
| 連線權限 | 只有知道指令 + 有金仙私鑰的人才能連 |
| 時效性 | 關掉視窗立刻斷線，不留後門 |
| 無殘留 | 不裝任何軟體，不留下任何服務 |
| 客戶端 | 只知道執行了 ssh.exe（Windows 內建） |

---

## ⚠️ 注意事項

- **需要 Windows 10 1809 以上**（內建 OpenSSH）
- **需要網路連線**（能連到 serveo.net 即可）
- **UAC 要按同意**（開 SSH Server 需要管理員權限）
- **防毒軟體可能擋** `start.bat`，加白名單即可
- **客戶電腦要有密碼**（SSH 登入需要 Windows 帳密）
