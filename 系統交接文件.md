- 母機 = 192.168.55.240，關機 = 所有 VM 全死
- **只用 API token**（openclaw@pve!vmops），不用 root 密碼
- 任何動到母機的操作需先提出申請，等老闆核准
- token 只能管理自己建立的 VM/CT

- MEMORY.md 只在主對話（Telegram）讀取
